YouTube Video Embed

Amalia.js with official iFrame player API from YouTube to play a video inside of Amalia.js.

Embedding Step-by-Step

Step 1 : Add youtube IFrame Player API 

<script src="https://www.youtube.com/iframe_api"></script>

Step 2 : Add amalia.js and third party libraries

<script type="text/javascript" src="bower_components/jquery/dist/jquery.js"></script>
<script type="text/javascript" src="bower_components/jquery-ui/jquery-ui.min.js"></script>
<script type="text/javascript" src="bower_components/raphael/raphael.js"></script>
<link href="build/css/amalia.js.min.css" type="text/css" rel="stylesheet"/>
<script src="build/js/amalia.js.min.js" type="text/javascript"></script>

Step 3 : Add youtube plugin

<script type="text/javascript" src="build/js/amalia.js-yt-player.min.js"></script>

Step 4 : Create a container for amalia.js player with specified id

   <div id="my-player-yt"></div>

Step 5 : Instantiate the player 

    $("#my-player-yt").mediaPlayer({
        player: "fr.ina.amalia.player.YtPlayer",
        src: 'NU8pS5rHmWM', // Youtube video ID
    });
